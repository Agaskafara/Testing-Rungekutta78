# Exercici 5

L'objectiu en aquest apartat és construir el retrat de fase de les equacions de Lorentz amb paràmetres $\sigma = 3$, $\rho = 26.5$ i $\beta = 1$ i amb les condicions inicials $(x_0, y_0, z_0) = (-0.416, -0.909, 0.014)$.

El codi associat es "scripts/ex5.py" i requereix:

* El path de la carpeta on es guarda l'output del programa "orbites.txt", passat com argument `--output_folder`. Per defecte, tenim la carpeta "outputs/ex5/orbites\_output".

Un cop obtingut el fixter "orbites.txt", ja podem generar el retrat de fase, l'objectiu de l'exercici. Per tal de generar-lo amb **gnuplot**, hem d'obrir un interpret de **gnuplot** en "ouputs/ex5" i executar les linies en el fitxer "outputs/ex5/comandes_gnuplot.txt".

Podem veure els resultats obtinguts en la carpeta "outputs/ex5/retrats\_de\_fase".
